# 🎫 Event Manager

Полнофункциональное приложение для управления мероприятиями с ролями пользователя и администратора.  
Бэкенд на **FastAPI**, фронтенд на **React + Vite + Max UI**.

---

## ✨ Возможности

### 👤 Для участников
- Просмотр списка активных событий с тегами и баллами
- Регистрация / отмена регистрации на мероприятие (в том числе свайпом)
- Личный профиль с баллами и настройками уведомлений

### 👑 Для администраторов
- Создание, редактирование и удаление событий
- Управление списком посетителей (поиск пользователей по нику, добавление/удаление)
- Назначение тегов через удобный селектор
- Архивация прошедших мероприятий

---

## 📁 Структура проекта


├── backend/ # FastAPI приложение
│ ├── main.py # основной файл с эндпоинтами
│ ├── models.py # Pydantic модели
│ └── dbs.py # тестовые данные (в памяти)
├── frontend/ # React (Vite + Max UI)
│ ├── src/
│ │ ├── api/ # типы и хелперы
│ │ ├── components/ # переиспользуемые компоненты
│ │ └── pages/ # страницы (профиль, события, админка)
│ ├── vite.config.ts
│ └── package.json
├── run.sh # скрипт запуска всего проекта
├── stop.sh # скрипт остановки
└── README.md
